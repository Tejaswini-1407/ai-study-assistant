# AI Study Assistant

⭐ AI Study Assistant is a full-stack AI-powered study platform that helps students learn faster from PDF materials by generating summaries, answers, quizzes, and flashcards.

## ✨ What It Does

AI Study Assistant focuses on making study materials easier to understand, revise, and practice.

Users can:

- Upload PDF study materials
- Generate clean AI summaries
- Ask questions from uploaded PDFs
- Create multiple-choice quizzes
- Generate flashcards
- Track quiz score and accuracy
- Save private study history
- View and delete saved study sets

## ⚡ Features

- 📄 PDF Upload
- 📝 AI Summary Generation

## ▶️ Run Locally

Clone the repository:

```bash
git clone https://github.com/Tejaswini-1407/ai-study-assistant.git
```

Go to the project folder:

```bash
cd ai-study-assistant
```

Install and start the backend:

```bash
cd server
npm install
npm run dev
```

Open a new terminal, then install and start the frontend:

```bash
cd client
npm install
npm run dev
```

Open the app:

```txt
http://localhost:5173
```

> Note: This project requires valid environment variables for Gemini API, MongoDB Atlas, JWT authentication, and frontend-backend connection.
